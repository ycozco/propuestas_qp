Backend & Lógica de Negocio

CakePHP 4.x Framework

  • ORM Robusto: Manejo eficiente de relaciones complejas (Olimpiadas -> Exámenes -> Preguntas).
  • Middleware de Autenticación: Seguridad integrada para protección de rutas.
  • Inyección de Dependencias: Código limpio y testable.
  • Plugins: Arquitectura modular separando lógica de 'Olimpiadas' del 'Núcleo'.
Base de Datos & Infraestructura

MySQL / MariaDB

  • Multi-Tenancy: Columna site_id en todas las tablas clave para separación lógica de sedes.
  • Integridad Referencial: Foreign Keys estrictas para evitar datos huérfanos.
  • Índices Optimizados: Búsquedas rápidas por DNI y Código de Estudiante.
APIs & Integraciones Externas

WhatsApp Business API

  • Envío de mensajes plantilla pre-aprobados por Meta.
  • Confirmación instantánea de pagos e inscripciones.
  • Notificación de resultados de exámenes.
