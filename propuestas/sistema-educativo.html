Visión del Sistema

Una plataforma completa que transforma la gestión educativa

Información Institucional

Portada web profesional con páginas institucionales, noticias, admisión y comunicados. CMS ligero para gestión de contenido.

Gestión de Olimpiadas

Temarios, banco de preguntas, exámenes programados, resultados en tiempo real e historial completo de participación.

Pagos Digitales

Inscripción con pago vía Yape, carga de voucher, verificación manual y confirmación automática por WhatsApp.

Registro Académico

Notas por bimestre y curso, consulta para estudiantes, carga para docentes, reportes exportables en PDF/Excel.

Multi-Sede

Operación en 3 sedes con datos filtrados por site_id, roles específicos y control de acceso granular.

Roles y Permisos (RBAC)

Flujo de Inscripción y Pago

Proceso robusto con confirmación por WhatsApp

1

Selección de Examen

El usuario elige el examen y completa el formulario de inscripción con sus datos personales.

→
2

Generación de Código

El sistema genera un código único de inscripción con monto e instrucciones para pago Yape.

→
3

Pago y Voucher

Usuario paga por Yape y sube imagen del voucher. Estado: "Pendiente de verificación".

→
4

Validación

Caja valida voucher, monto y coincidencia de datos. Marca pago como "Confirmado".

→
5

Confirmación WhatsApp

Se confirma inscripción y se envía mensaje automático por WhatsApp al estudiante.

Stack Tecnológico

Tecnologías modernas y probadas en producción

CakePHP 4.x

Framework PHP robusto con ORM, autenticación y autorización integrada

MySQL/MariaDB

Base de datos relacional con soporte completo para transacciones

WhatsApp Business API

Integración con Meta o Twilio para confirmaciones automáticas

mPDF / PHPSpreadsheet

Generación de reportes en PDF y exportación a Excel

RBAC + Auditoría

Control de acceso basado en roles con trazabilidad completa

Multi-Tenant por site_id

Arquitectura multi-sede con filtrado automático por sede
